5.10 Random Numbers

Notes

Random numbers are useful in software development for many reasons-games mechanics, generating usernames/passwords, anytime you need something seemingly unpredictable

Random numbers in JS

javaskript provides a number of functions that are useful for generating random numbers

  • math.randomis a function that returns a pseudo-random number between 0 and 1, not including 1
  • math.flooris a function that takes a decimal number and rounds it downn-chops off the decimal part
